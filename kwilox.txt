Kwilox is a relaxation centre located in Abeokuta, Ogun State. It is known for
selling expensive drinks and confectionery. Create an API endpoint for the
following cases
Users:
1. Can register with the following data Phone Number, Age, Email, First Name, Last Name and Password
2. Can Login using Email address and Password
3. Can see all available items (drinks and confectionery) uploaded by Admin.
Admin:
• Can see all registered users
• Can search for users using their email address.
• Can post a list of available items (drinks and confectionery) so users logged in
can see.
